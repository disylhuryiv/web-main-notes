# web-main-notes

本仓库用于归档和发布多个独立 HTML 页面。这些页面以静态文件形式存放，可直接通过浏览器访问，通常用于记录、展示或测试特定功能模块。

## 项目简介

`web-main-notes` 是一个轻量化的页面集合仓库，不针对任何特定域名或网站。仓库内每个 HTML 文件均为独立单元，彼此之间无强依赖关系，便于单独查看、分享或嵌入。

## 目录说明

```
web-main-notes/
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面引用的样式、脚本、图片等资源
├── README.md       # 本文件
└── .gitignore      # Git 忽略规则
```

- `pages/`：核心目录，每个 HTML 文件代表一个独立页面。
- `assets/`：存放页面所需的公共资源，如 CSS、JavaScript、字体或图标。

## 页面归档说明

每个页面均以独立文件形式归档，文件名应简洁且反映页面用途。归档时建议遵循以下原则：

- 每个页面应自包含，尽可能减少对外部资源的依赖。
- 如需引用外部资源，请优先使用相对路径指向 `assets/` 目录。
- 页面内容应保持通用性，不包含任何特定域名或商业信息。

## 维护说明

- 新增页面：直接在 `pages/` 目录下创建新的 HTML 文件，并确保无误后提交。
- 更新页面：修改对应文件后，更新相关注释或版本标注（如有）。
- 资源更新：如需修改公共资源，请同步检查所有引用该资源的页面。
- 删除页面：移除文件前，请确认无其他页面或外部引用。

## 注意事项

- 本仓库仅用于存放静态 HTML 页面，不涉及后端逻辑或数据库。
- 请勿在页面中包含任何形式的营销、诱导点击或第三方跟踪代码。
- 提交前请确保页面在不同浏览器中基本兼容。
